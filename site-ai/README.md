# MoteurDirect AI

Prototype de site web public pour vendre des moteurs usages avec assistant IA visuel.

## Lancer en local

```powershell
cd site-ai
npm install
npm run dev
```

Puis ouvrir http://localhost:3000

## Objectif

- Page d'accueil premium
- Assistant IA vocal/chat en simulation MVP
- Formulaire de demande moteur
- Structure prete pour OpenAI Realtime, VendorCompagnon et Pipedrive

## Prochaine etape

Connecter le formulaire a une API qui envoie les leads dans VendorCompagnon ou Pipedrive.
